## Step by step guide to view the slides locally

This assumes that you have go compiler and git installed on your system.

```
git clone https://github.com/Nisarg2061/Presentations.git
```
```
cd Presentations
```
Option 1: Using make
```
make present
```
Option 2: using bash script
```
./present.sh
```
